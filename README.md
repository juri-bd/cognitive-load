# cognitive-load
AI-powered UI cognitive load analyzer that evaluates screenshots for visual complexity, hierarchy, contrast, and usability friction.

Analyze UI screenshots using AI to estimate cognitive load, visual complexity, hierarchy issues, and usability friction.

## Features

- Screenshot upload
- AI-powered UI analysis
- Cognitive load scoring
- Visual hierarchy evaluation
- Contrast and readability analysis
- Evidence-based findings
- Actionable UX recommendations

- ## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- OpenAI Vision API
- Vercel
